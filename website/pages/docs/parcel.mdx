# Parcel

[Parcel](https://parceljs.org/) provides an official plugin to import SVG as React components.

<carbon-ad />

SVGR is natively supported by [Parcel v2+](https://parceljs.org/). Please report to ["Importing SVG as a React component" guide](https://parceljs.org/languages/svg/#importing-as-a-react-component) on official parcel documentation.

> If you still use Parcel v1, [`@svgr/parcel-plugin-svgr`](https://www.npmjs.com/package/@svgr/parcel-plugin-svgr) is deprecated and only compatible with Parcel v1.
